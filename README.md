# Sol Bounties Frontend (Next.js)

## Local
1) `cp .env.example .env.local`
2) Ajusta `NEXT_PUBLIC_API_URL` al backend (ej: http://localhost:3001)
3) `npm install`
4) `npm run dev`

## Badges de Canva
- Copia tus PNG a `public/badges/`
- Renombra cada archivo para que coincida con el `badge.key` del backend:
  - `wallet_verified.png`, `first_bounty_win.png`, etc.
